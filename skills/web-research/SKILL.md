# Web Research

Gather reliable information and cite it.

1. **Clarify the question** and identify what specific facts you need to answer it.
2. **Fetch** relevant pages with the `web_fetch` tool. Prefer official documentation, primary sources, and reputable references.
3. **Cross-check** important claims against a second source when accuracy matters.
4. **Extract** only the relevant facts; ignore navigation, ads, and boilerplate.
5. **Synthesize** a clear answer in your own words.
6. **Cite** the source URLs you used so the user can verify.

## Guidelines

- Prefer primary and official sources over aggregators.
- Note the publication date; prefer current information for fast-moving topics.
- Distinguish established facts from opinion or speculation.
- If sources conflict, say so and explain the discrepancy rather than picking silently.
- If you cannot verify a claim, state that plainly instead of guessing.
